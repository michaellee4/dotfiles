# dotfiles

My personal macOS dotfiles, managed with symlinks.

## Install

```sh
git clone git@github.com:michaellee4/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## Configs

| Tool | Description |
|------|-------------|
| [Aerospace](https://github.com/nikitabobko/AeroSpace) | i3-like tiling window manager for macOS |
| [Kitty](https://github.com/kovidgoyal/kitty) | Fast, feature-rich GPU-based terminal emulator |
| [Neovim](https://github.com/neovim/neovim) | Hyperextensible Vim-based text editor |
| [Yazi](https://github.com/sxyazi/yazi) | Blazing fast terminal file manager |
| [Sketchybar](https://github.com/FelixKratz/SketchyBar) | Highly customizable macOS status bar replacement |
| [Starship](https://github.com/starship/starship) | Minimal, fast, customizable shell prompt |
| [Borders](https://github.com/FelixKratz/JankyBorders) | Adds colored borders to focused windows |
| [AutoRaise](https://github.com/sbmpost/AutoRaise) | Auto-raises and focuses windows on mouse hover |
| [Git](https://github.com/git/git) | Global git ignore rules |
| Zsh | Shell config (`~/.zshrc`, `~/.zprofile`) |
